⚙ 共有設定（家族で同期）

下の設定を入れると、家族みんなのPC・iPhoneで同じ予定・準備物が自動で同期されます。
まだ設定が無くてもアプリはこの端末だけで使えます（ローカル保存）。

大文字・スペースは自動で統一されます（Ichika Riku→ichika-riku）。半角英数字とハイフンがおすすめ。

▶ Firebaseの設定手順を表示

  1. Firebaseコンソールを開き「プロジェクトを追加」
  2. 左メニュー「構築 → Firestore Database」→「データベースを作成」→テストモードで開始
  3. プロジェクト設定（⚙）→「マイアプリ」→ </>（ウェブ）でアプリを登録
  4. 表示される firebaseConfig = { ... } の { ... } 部分を上の①に貼り付け
  5. ②に家族共通の合言葉（共有ID）を入力 → 保存
  6. 同じ①②を家族の端末にも入れれば全員同期されます

※テストモードは初期は誰でも読み書き可。気になる場合は後でセキュリティルールで共有IDを限定できます（手順は作成者にご相談ください）。
